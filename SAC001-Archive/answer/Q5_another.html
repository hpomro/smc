Q5 -Cat and Stairs

猫のゴリズム君が全8段の階段を上る。
ただし,ゴリズム君は一回の動作で 1段または2段または3段 の階段を上ることができる。
ただし,以下の動作はできない。

  • 6段目にいるときに3段上る。
  • 7段目にいるときに3段上る。
  • 7段目にいるときに2段上る。

    • 最初ゴリズム君が0段目にいるとき,8段の階段を上り切る動作の方法は何通りあるか。ただし,通過する段があってもよい。

    Answer

     81 

    Key

    場合の数

    Description

    1段上る動作をk回,2段上る動作をl回,3段上る動作をm回行うとする。

    (k,l,m)の値の組に対する上り方は (k+l+m)! k!⁢ l!⁢ m! 通りある。

    k+2⁢l+3⁢m=8

    (1) m=0のとき
    k+2⁢l=8
    これを解いて(k,l)=(8,0),(6,1),(4,2),(2,3),(0,4)
    よってm=の0ときの上り方は,
    (8+0)! 8!⁢ 0! + (6+1)! 6!⁢ 1! + (4+2)! 4!⁢ 2! + (2+3)! 2!⁢ 3! + (0+4)! 0!⁢ 4! = 1+7+15+10+1=34

    (2) m=1のとき
    k+2⁢l=5
    これを解いて(k,l)=(5,0),(3,1),(1,2)
    よってm=1のときの上り方は,
    (5+0+1)! 5!⁢ 0! + (3+1+1)! 3!⁢ 1! + (1+2+1)! 1!⁢ 2! = 6+20+12=38

    (3) m=2のとき
    k+2⁢l=2
    これを解いて(k,l)=(0,1),(2,0)
    よってm=2のときの上り方は,
    (0+1+2)! 0!⁢ 1!⁢ 2! + (2+0+2)! 2!⁢ 0!⁢ 2! = 3+6=9

    以上から,求める場合の数は, 34+38+9=81 (通り)。

    (答) 81(通り)
