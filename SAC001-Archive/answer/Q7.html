Q7 -Sum of Sums of Divisors

正の整数nの正の約数の総和をf(n)とする。
f(2025)+f(2・2025)+f(3・2025)+・・・+f(9・2025)+f(10・2025) の値を求めよ。

Answer

 282057 

Key

まとめて計算

Description

正整数n,mが互いに素であるとき, f(nm)= f(n)⁢ f(m)

pが素数,kが正整数のとき, f(pk) = p0+ p1+ p2+ ・・・ + pk

よって, f(2025) = f(34) ⁢ f(52) = 121×31=1751


f(2025×1)=f(2025)>

f(2025×2)=f(2025)⁢f(2)=3f(2025)

f(2025×3)=f(2025)×f(35)f(34)

f(2025×4)=f(2025)⁢f(22)=7f(2025)

f(2025×5)=f(2025)×f(53)f(52)

f(2025×6)=f(2025)⁢f(2)×f(35)f(34) =3f(2025)×f(35)f(34)

f(2025×7)=f(2025)⁢f(7)=8f(2025)

f(2025×8)=f(2025)⁢f(23)=15f(2025)

f(2025×9)=f(2025)×f(36)f(34)

f(2025×10)=f(2025)⁢f(2)×f(53)f(52) =3f(2025) ×f(53)f(52)


よって,(与式) = f(2025) × { 34+ f(35)f(34) ×4+ f(36)f(34) + f(53)f(52) ×4 }


f(35)f(34) = 1+ 35f(34)

f(36)f(34) = 1+ 35+36f(34)

f(35)f(34) ×4+ f(36)f(34) = 5+ 35(4+1+3)f(34) = 5+ 1944f(34)

f(53)f(52) = 1+ 53f(52)

f(53)f(52) ×4= 4+ 500f(52)


したがって,(与式) =f(34) ⁢ f(52) × { 43+ 1944f(34) + 500f(52) }

= 121×31×43+ 31×1944+ 121×500

=121×1833 +31×1944

=121×1833 +31×1833 +31×111

=1833×152+31×111

=278616+3441

=282057

(答) 282057
